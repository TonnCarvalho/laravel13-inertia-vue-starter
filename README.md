
### Install
1. ``composer install``
2. `npm install`
3. Duplique o arquivo **.env.example** e renomei para **.env**
4. Crie o banco de dados no seu SGBD.
5. `php artisan migrate`
6. `php artisan key:generate`
7. `php artisan serve`
8. `npm run dev`
