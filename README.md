
### Install
1. ```composer install```
2. ```npm instal```
3. Duplique o arquivo **.env.example** e renomei para **.env**
4. Crie o banco de dados no seu SGBD.
5. ```php artisan migrate```
6. ```php artisan key:generate```
7. ```php artisan serve```
8. ```npm run dev```


### Pacotes
[Laravel PT BR](https://github.com/lucascudo/laravel-pt-BR-localization)<br>
Módulo de linguagem pt-BR (português brasileiro) para Laravel.<br>
[Laravel Debugbar](https://github.com/fruitcake/laravel-debugbar)<br>
Pacote integra a barra de depuração PHP com o Laravel.<br>
[ZiggyVue](https://github.com/tighten/ziggy)<br>
Facilitando o uso de rotas nomeadas do Laravel em JavaScript. `route('route.name')`<br>
